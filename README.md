# Waste Management & Recycling Process Explainer Bot

## Description
An AI-powered educational chatbot that explains waste segregation, recycling, composting, and hazardous waste handling using Google Gemini Flash models.

## Features
- Explains waste management workflows
- Simple, citizen-friendly responses
- No service scheduling or enforcement actions
- Streamlit-based public interface

## Tech Stack
- Google Gemini Flash
- Python
- Streamlit

## How to Run
1. Install dependencies  
   pip install -r requirements.txt

2. Set Google API Key  
   export GOOGLE_API_KEY="your_api_key"

3. Run the app  
   streamlit run app.py
